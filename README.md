# Bulk Questionnaire Upload

## 🧪 Key Features

- ✅ Upload XLS/XLSX form templates  
- ✅ Live form preview with auto-mapped components  
- ✅ Inline error handling for unsupported formats  
- ✅ Edit & download forms using Excel  
- ✅ Support for large files and complex question types
